Networking

Time to connect things to the internet!

How do I connect to the internet?

The ESP8266 microcontroller has a built-in WiFi radio, so it is a very easy to integrate internet connectivity into your project! The Arduino Board Package for the ESP8266 comes with a good set of libraries to get you started. You can find them in Sketch->Add Library->ESP8266... The one we use for our examples is ESP8266WiFi.

We also found a good JSON parsing library for Arduino. It's used in the following networking example, so you should download it! The instructions are on the basics page.

This example project (this link) runs through some of the basics. It will connect to this website, attempt to load "/ws2812-test.html". This page has some data to be parsed. The JSON library runs through it, and picks out colors to write to the LEDs. If someone changes the page (like in the middle of a workshop), the board will reload the page and update the LEDs to new colors.
